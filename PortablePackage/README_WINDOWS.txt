Windows Portable Package
=========================

This folder is for a zip-friendly portable build.

1) Build the EXE in Visual Studio (Release x64).
   Output path:
   Builds\VisualStudio2022\x64\Release\Standalone Plugin\StaticCurrentsPlugin.exe

2) Copy the EXE into:
   PortablePackage\App\StaticCurrentsPlugin.exe

3) Zip the PortablePackage folder and distribute it.
