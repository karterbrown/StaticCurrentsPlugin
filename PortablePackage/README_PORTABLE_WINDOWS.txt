Static Currents Plugin - Portable Windows Build
===============================================

1) Install Microsoft Visual C++ Redistributable (x64) 2015-2022
   https://aka.ms/vs/17/release/vc_redist.x64.exe

2) Unzip this package

3) Run:
   App\StaticCurrentsPlugin.exe

If the app does not start, confirm the VC++ Redistributable is installed
and try again.
